# za38-cli 实现计划（v2 — 基于 dcode 功能裁剪）

> **给执行者：** 使用 compose:subagent（推荐）或 compose:execute 按任务逐个实现。步骤使用复选框（`- [ ]`）跟踪进度。

**目标：** 构建企业级编码智能体 CLI，opentui（TS/Node）表现层 + deepagents（Python）agent 内核，通过 JSON-RPC over stdio 通信，单 npm 包分发。核心功能参照 dcode，保留全部 Core 功能 + JS 解释器，裁剪沙箱/目标评分/更新系统/LangSmith/通知等。

**架构：** 薄协议 + 智能内核。Node/Bun 将 Python 作为子进程启动。IPC 是换行分隔的 JSON-RPC 2.0，走 stdin/stdout。Python 拥有全部 agent 逻辑（deepagents + LangGraph）。Node 通过 opentui + React 渲染 TUI。

**技术栈：** TypeScript（Bun 为主，兼容 Node 22+）、@opentui/core + @opentui/react（vendor 引入）、React 19、Python 3.12、deepagents、LangChain、LangGraph、langchain-quickjs（JS 解释器）、uv（Python 包管理器）

## 全局约束

- Python >=3.11（deepagents 要求），通过 uv 目标 3.12
- 运行时：Bun >=1.3.0（主力，opentui 原生 FFI）；Node 22+ 加 `--experimental-ffi` 也可用
- @opentui/core 已发布到 npm（^0.4.3）；@opentui/react 私有 —— vendor 引入
- stdout 仅供 JSON-RPC 协议消息；Python 调试输出走 stderr
- JSON-RPC 消息为换行分隔的 UTF-8 JSON，每行一个对象
- deepagents >=0.6.12，langchain >=1.3.12，langchain-quickjs >=0.3.2
- npm 包名：`@za38/cli`，二进制名：`za38`
- Python 包名：`za38-agent`，导入名：`za38_agent`
- 测试：pytest（Python，用 `GenericFakeChatModel` mock 模型）、bun test（Node/TS）
- **原则**：dcode 已有的代码可直接照搬，裁剪不需要的功能

- [ ] **步骤 4：实现 create_za38_agent**

参照 dcode `agent.py:create_cli_agent()` (line 1409-1976)，裁剪后的核心结构：

```python
# packages/agent/za38_agent/agent.py
"""za38 agent 内核 —— 参照 dcode create_cli_agent 裁剪版。"""
from pathlib import Path
from typing import Any, Sequence

from deepagents import create_deep_agent
from deepagents.backends import LocalShellBackend
from langchain_core.language_models import BaseChatModel
from langchain_core.tools import BaseTool
from langgraph.checkpoint.memory import MemorySaver

# 照搬的 dcode 模块
from za38_agent.ask_user import AskUserMiddleware
from za38_agent.local_context import LocalContextMiddleware
from za38_agent.shell_allow_list import ShellAllowListMiddleware

_PROMPT_PATH = Path(__file__).parent / "prompts" / "system_prompt.md"

def _load_system_prompt() -> str:
    return _PROMPT_PATH.read_text(encoding="utf-8")


def create_za38_agent(
    model: BaseChatModel | str,
    assistant_id: str = "za38",
    *,
    tools: Sequence[BaseTool | Any] | None = None,
    system_prompt: str | None = None,
    interactive: bool = True,
    auto_approve: bool = False,
    shell_allow_list: list[str] | None = None,
    enable_ask_user: bool = True,
    enable_memory: bool = True,
    enable_skills: bool = True,
    enable_interpreter: bool = True,  # 默认开启（dcode 默认关闭）
    checkpointer: Any = None,
    mcp_server_info: list | None = None,
    cwd: str | None = None,
    workdir: str | None = None,
) -> Any:
    """创建 za38 编码 agent。

    参照 dcode create_cli_agent，裁剪沙箱/评分/远程异步子 agent。
    """
    from za38_agent.providers.za38_gateway import resolve_model as _resolve
    resolved_model = _resolve(model) if isinstance(model, str) else model

    backend = LocalShellBackend(root_dir=workdir or cwd or ".", virtual_mode=False)
    agent_middleware: list = []

    # 1. ConfigurableModelMiddleware（运行时模型切换）
    # TODO: 从 dcode 照搬 configurable_model.py

    # 2. AskUserMiddleware（交互式提问，仅 interactive 模式）
    if interactive and enable_ask_user:
        agent_middleware.append(AskUserMiddleware())

    # 3. MemoryMiddleware（AGENTS.md 记忆）
    if enable_memory:
        from deepagents import MemoryMiddleware
        agent_middleware.append(MemoryMiddleware(sources=["~/.za38/AGENTS.md", "./AGENTS.md"]))
        # ManagedMemoryGuardMiddleware 照搬 dcode

    # 4. SkillsMiddleware（技能加载）
    if enable_skills:
        from deepagents import SkillsMiddleware
        agent_middleware.append(SkillsMiddleware(skills_dirs=[...]))  # 参照 dcode 7 源目录

    # 5. CodeInterpreterMiddleware（JS 解释器）
    if enable_interpreter:
        from langchain_quickjs import CodeInterpreterMiddleware
        from langchain_core._api import suppress_langchain_beta_warning
        with suppress_langchain_beta_warning():
            agent_middleware.append(CodeInterpreterMiddleware(
                tool_name="js_eval",
                timeout=30,
                memory_limit=128 * 1024 * 1024,
                max_ptc_calls=50,
                max_result_chars=50000,
                ptc=None,  # safe 模式
            ))

    # 6. LocalContextMiddleware（git 状态/项目结构注入）
    agent_middleware.append(LocalContextMiddleware(backend=backend, mcp_server_info=mcp_server_info))

    # 7. ShellAllowListMiddleware（shell 白名单）
    if shell_allow_list:
        agent_middleware.append(ShellAllowListMiddleware(shell_allow_list))

    # 8. HITL（interrupt_on）—— 参照 dcode _add_interrupt_on() 裁剪
    interrupt_on = _add_interrupt_on(auto_approve=auto_approve, interactive=interactive)

    # 9. SummarizationToolMiddleware（compact_conversation 工具）
    from deepagents import SummarizationToolMiddleware
    agent_middleware.append(SummarizationToolMiddleware())

    prompt = system_prompt or _load_system_prompt()
    all_tools = list(tools) if tools else []

    return create_deep_agent(
        model=resolved_model,
        tools=all_tools,
        middleware=agent_middleware,
        backend=backend,
        system_prompt=prompt,
        interrupt_on=interrupt_on if not auto_approve else None,
        checkpointer=checkpointer or MemorySaver(),
    )


def _add_interrupt_on(*, auto_approve: bool = False, interactive: bool = True) -> dict:
    """参照 dcode agent.py:_add_interrupt_on (line 1301-1389) 裁剪版。

    裁剪：web_search, fetch_url, start_async_task, update_async_task, cancel_async_task
    保留：execute, write_file, edit_file, delete, task, compact_conversation
    """
```
