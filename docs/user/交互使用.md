# 交互使用

启动 `za38`（源码开发阶段可用 `bun run dev --`）后，直接输入编码任务。Harness Code 会将流式文本、工具调用、审批请求和问题显示在同一会话时间线中。

## 快捷键

| 快捷键 | 行为 |
| --- | --- |
| `Enter` | 发送输入 |
| `Shift+Enter` | 输入换行 |
| `/` 或 `Ctrl+P` | 打开本地 Slash Command 菜单 |
| `Esc` | 关闭命令菜单或取消当前运行 |
| `Ctrl+C` | 优先清空输入，其次取消运行，最后退出 |
| `Ctrl+O` | 展开或折叠工具详情 |
| 空输入时 `↑` / `↓` | 回填历史提示词；没有历史时浏览会话 |
| 空输入时 `PageUp` / `PageDown` | 按半页浏览会话 |

## 已实现的 Slash Command

- `/help`：显示命令说明。
- `/clear`：清空当前会话并回到首页。
- `/force-clear`：取消执行后清空当前会话。
- `/status`：显示工作区、模型、执行后端与审批模式。
- `/version`：显示 CLI 版本。
- `/quit` 或 `/q`：退出。

`/threads`、`/mcp`、`/tools`、`/agents`、`/remember` 和技能选择尚未接入后端；不要将它们当作可用命令。

模型配置、无头调用和配置查询见 [模型配置](模型配置.md) 与 [快速开始](快速开始.md)。
