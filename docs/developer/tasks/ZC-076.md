## 范围

- `/usage` 展示 Provider 实际上报的 input/output/cache/reasoning token、工具调用、运行耗时和压缩节省；没有可信数据时显示 unknown，不推算费用。
- `/history` 提供当前 project/thread 的用户 Prompt 与运行摘要搜索，支持打开对应安全摘要但不显示隐藏的敏感事件 payload。
- `/copy` 复制最近一条可复制 Agent 输出或指定 Viewer 片段；终端/系统剪贴板不可用时给出可操作提示。
- 与 ZC-012 的完整 thread 导出共用访问与保留策略，避免再创建第二份历史存储。

## 非范围

- 不实现跨项目全文索引、云端分析、成本估算、数据上传、共享链接或隐藏 Prompt 的导出。
- 不改变 Provider usage 原始计量、Thread 删除/导出策略或 EventLog 回放语义。

## 验收清单

- 多模型、多 run、压缩、缺失 usage、损坏事件和恢复后的统计不会串 thread，unknown 状态稳定。
- 历史搜索、复制、权限错误和窄终端操作有端到端回归，任何查询均不暴露其他 project 的内容。
- 文档明确列出数据来源与不支持项，敏感字段 redaction 有测试。
