## 范围

扩展 ZC-016 的 MCP session manager，使其管理 HTTP 与 SSE 连接并遵守 ZC-005 的信任决定。

## 非范围

不实现 OAuth、自动重连 UI、任意 URL 直连或用户自定义 TLS 绕过。

## 验收清单

- HTTP 和 SSE mock server 均有工具注册、调用和 cleanup E2E。
- 认证头仅从显式白名单环境变量插值，绝不回显秘密。
- 错误状态可被 `mcp.list` 读取但不会暴露敏感 URL 查询或 header。
