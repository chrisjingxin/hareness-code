## 范围

定义远端子 Agent 注册配置和最小 task 生命周期，使用现有 v2 `source` 字段标识子 Agent 事件。

## 非范围

不允许模型传入任意远端 URL，不实现多租户队列、跨任务共享文件系统或子 Agent 自主提权。

不实现 Agent Team 的持久 Mailbox、Coordinator 或并行调度；远端子 Agent 后续只能以 ZC-062 定义的受控 worker 适配器接入。

## 验收清单

- 仅显式登记的端点可作为子 Agent 类型调用。
- 启动、进度、完成、失败、取消和结果截断都有 mock E2E。
- 主 Agent、远端子 Agent 的模型凭据和执行权限相互隔离。
