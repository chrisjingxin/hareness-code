## 范围

> 历史说明：该 0.1.74 快照和自建 native artifact 已由 ZC-046 的企业 OpenTUI 0.4.3 React 方案替换，不再属于当前交付物；本任务证据仅保留原迁移审计记录。

- 使用企业 `@opentui/solid` 0.1.74 改造现有 React TUI，并复用当前 JSON-RPC、状态归约和纯 UI 算法。
- 固定 OpenTUI Core 0.1.74 上游源码快照，生成并校验 macOS arm64/x64、Linux x64 glibc、Windows x64 native artifact。
- 使用企业 Python 版本更新 `pyproject.toml` 和 `uv.lock`，移除 QuickJS/js_eval，保留 MCP adapter。

## 非范围

- 不交付完整的 macOS、Linux、Windows 自包含运行包或安装器；这些仍由 ZC-007、ZC-024、ZC-026 负责。
- 不实现 MCP 工具加载、模型 Profile、远端 sandbox 或协议版本变更。
- 不支持 Linux musl、Linux arm64 或 Windows arm64 OpenTUI native artifact。

## 验收清单

- Solid TUI 覆盖首页、会话、审批、问答、取消、Slash 命令和窄终端降级。
- Core native manifest 包含四个目标的来源、版本、SHA-256 和许可证信息；不支持目标有稳定诊断。
- 旧 Core 缺少语法 parser API 时，代码块安全降级为纯文本。
- `uv lock --check`、Python/Bun 回归、mock 网关 E2E 和目标平台 artifact 冒烟测试通过。
