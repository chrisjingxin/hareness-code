## 范围

实现子 Agent 定义解析、来源优先级、受限模型继承和列表 RPC；保留 SDK 的 `general-purpose` 作为兜底子 Agent。

## 非范围

不实现异步远端子 Agent、子 Agent 独立 sandbox、TUI 面板或跨进程任务队列。

不建立 Agent Team 调度或按 thread 缓存子 Agent 图；自定义定义必须作为 Runtime Profile 的静态能力参与图构建。

## 验收清单

- 有效定义能出现在 `agents.list` 并传入 `create_deep_agent`。
- 缺描述、非法 front matter、路径逃逸和重复名称有稳定诊断。
- 项目覆盖用户、默认模型继承和子 Agent 调用具有 Agent 工厂测试。
