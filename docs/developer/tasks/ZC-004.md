## 范围

提供 `~/.harness/AGENTS.md` 与 `<workspace>/.harness/AGENTS.md` 的安全发现和加载，并明确项目级来源相对用户级来源的优先级。

## 非范围

不实现 Harness 管理记忆区块保护、Skills、子 Agent、`/remember` 命令或远程记忆同步。

## 验收清单

- 用户级与项目级来源的优先级和缺失文件降级有测试。
- symlink、不可读文件和工作区外项目来源不会注入模型上下文。
- 加载顺序与实际注入内容有 Agent 工厂回归测试。
