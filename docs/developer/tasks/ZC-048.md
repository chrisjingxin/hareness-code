## 范围

- 扫描 `<workspace>/.harness/skills/`、`~/.harness/skills/local/`、打包内置目录和市场安装目录。
- 使用 canonical ID、SHA-256 snapshot、front matter 校验和受限 UTF-8 资源读取；正文和资源通过 `/.harness/skills/<canonical-id>/...` 虚拟只读路径由 `read_file` 渐进式读取。
- 通过 JSON-RPC `skills.*` 方法、`skill.loaded` 领域事件和 `run.start.requested_skill` 连接 Python Agent、CLI 和 TUI。
- 提供 `harness.skill_marketplaces` entry point，企业 Provider 负责认证、目录和 artifact 获取。

## 非范围

- 不执行 Skill 自动 Hook、模型覆盖、子 Agent、MCP 连接或任意 `allowed-tools` 声明。
- 不连接公网或硬编码企业市场地址、凭据；远端 sandbox 仍禁用宿主 Skill。
- 不引入完整 wire replay；本次只固定 snapshot ID 和加载事件。
