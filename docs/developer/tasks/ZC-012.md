## 范围

在 ZC-003 的 SQLite 基础上保存 thread 的工作区、创建/更新时间、首条提示词摘要和模型摘要，并暴露只读列表与恢复能力。

## 非范围

不做云同步、线程搜索、删除、导出、跨工作区恢复或会话侧栏重设计。

## 验收清单

- `threads.list` 只返回请求工作区范围内的有限条目。
- `za38 threads list` 与 TUI 恢复入口展示相同稳定 metadata。
- 线程恢复、缺失 ID、损坏记录和并发读取有端到端测试。
