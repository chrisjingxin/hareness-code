## 范围

在 ZC-003 的 SQLite 基础上增加用户明确需要的 thread 生命周期管理能力，并保持 project/thread/message 领域模型。

## 非范围

不做云同步、跨 project 恢复或重新引入 session 概念。

## 验收清单

- 删除、导出或保留策略必须显式确认，并只作用于当前 project 的 thread。
- 新入口与现有 `/resume` 选择器共享稳定摘要，不要求用户输入 `thread_id`。
- 生命周期操作、损坏记录和并发读取有端到端测试。
