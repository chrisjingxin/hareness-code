## 范围

在已有 checkpoint 与协议事件之间增加有限持久事件存储，并通过能力协商提供只读查询/回放。

事件记录必须绑定实际执行使用的 Runtime Profile 摘要；不得依赖已被淘汰的 Agent 图保存回放状态。Agent Team 完成后，ZC-062 仅可将受限 team 生命周期投影到本任务的 EventLog，不能把内部 mailbox 原文直接写入回放。

## 非范围

不实现任意客户端的 steer、跨设备实时同步、无限历史保存或将原始工具大输出完整持久化。

## 验收清单

- 文本、工具摘要、完成、取消和失败事件按连续 sequence 回放。
- 已解决的审批/问答只以 `interaction.resolved` 回放，不重新创建 request。
- 重复、缺口、截断和过期清理都有协议与 TUI 回归测试。
