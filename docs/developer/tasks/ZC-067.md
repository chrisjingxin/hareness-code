## 范围

- 定义模型 Profile 的只读列表与线程选择交互所需的版本化 DTO/RPC，显示名称、支持能力、上下文窗口、当前/不可用状态及脱敏来源。
- `/model` 无参数打开 Picker，参数作为查询；唯一合法匹配经过明确选择后写入下一次运行的 thread 配置。
- 对已绑定 Runtime Profile 的 thread 显示不可变绑定和新建 Thread 引导，防止静默修改历史 thread 的模型身份。
- 使用 Command Registry 的 capability、空闲与确认策略控制菜单可见性和执行状态。

## 非范围

- 不接入新 Provider、账号登录、费用自动优化或运行中模型切换。
- 不绕过 ZC-019 的 API Key 来源、RuntimeProfileKey 或恢复策略。

## 验收清单

- 无配置、未知查询、能力不足、多个匹配、已绑定 thread 和活动 run 均有清晰 TUI 状态与协议测试。
- 选择后实际 run 使用记录的 Profile，恢复 thread 不会被 Picker 默默改写，诊断和 UI 不含秘密。
- Picker 键盘、窄终端、取消和 RPC 失败回归通过。
