## 范围

实现独立的 MemoryGuard middleware，保护由 Harness 维护的标记区块，并在模型不可见 marker 时仍保持区块完整。

## 非范围

不自动向 AGENTS.md 写入用户偏好，不保护用户自行定义的任意文本，也不实现文件版本控制。

## 验收清单

- write_file、edit_file 和 delete 对管理区块的破坏均被拒绝或恢复。
- marker 缺失、文件不可读、目录删除和 symlink 情况 fail closed。
- 非管理内容的合法编辑与既有 HITL 行为不回归。
