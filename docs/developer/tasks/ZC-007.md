## 范围

建立第一个可验证的平台发行物，为 Linux、Windows 和安装器任务提供稳定的 manifest 格式。

## 非范围

不实现 Linux、Windows、在线安装脚本、自动更新或代码签名。

## 验收清单

- 运行包不依赖开发工作区、`PYTHONPATH` 或用户安装的 Python。
- manifest 枚举每个包含资产及其 SHA-256。
- CI 或本地干净目录冒烟测试覆盖 `harness --version`、配置读取和 mock SSE 单轮执行。
