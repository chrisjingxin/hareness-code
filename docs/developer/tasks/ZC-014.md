## 范围

建立 Harness 原生 Skill 目录约定与加载器，来源优先级固定为内置、用户级、项目级，后者可以按名称覆盖前者。

## 非范围

不支持 `.claude`、`.agents` 兼容目录、远程下载、插件市场或手工 `/skill:<name>` UI。

## 验收清单

- 合法、缺失、front matter 非法、同名覆盖和 symlink 攻击均有测试。
- 仅成功加载的 Skill 出现在 `skills.list` 结果中。
- 内置 za38 Skill 的来源、版本和安全边界有开发者文档。
