## 范围

- 借鉴 MiMo Code 的 dialog 布局，把 Skill 搜索和两列目录显示为会话上的居中浮层。
- 在不改变 Skill JSON-RPC、选择状态或 /skill 执行行为的前提下更新 Slash 文案与用户文档。

## 非范围

- 不新增 Skill 分类、安装、市场连接或多选。
- 不复制 MiMo Code 的组件、依赖或主题。

## 验收清单

- 选择器不会以背景色清空会话画布；紧凑终端安全降级。
- /skills 与 /skill:<id> 的选择行为维持不变；/skill <id> [args] 直接发送 run.start。
- OpenTUI 渲染、类型和项目一致性检查通过。
