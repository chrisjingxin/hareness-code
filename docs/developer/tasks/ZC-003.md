## 验收清单

- 进程重启后，使用同一 `thread_id` 的下一轮能读取此前消息。
- SQLite 连接、关闭和损坏数据库的失败路径有 Python 回归测试。
- 文档说明本地数据位置、保留范围和手动清理方式。
