## 范围

为 Windows 打包 Python sidecar、Node/Bun 启动层和 OpenTUI native 资源，实现 PowerShell 与 CMD 安装入口。

## 非范围

不实现 Windows 自动更新、MSI、企业软件中心集成或跨架构 Windows 支持。

## 验收清单

- PowerShell 和 CMD 安装流程均校验 artifact hash，失败不破坏现有安装。
- 安装后 `harness --version`、配置读取和 mock SSE 单轮执行通过。
- 对最终用户只暴露 `harness` 命令；`za38` 仅保留为内部框架、包和配置命名，不得作为安装后的公开入口。
- 路径空格、权限不足和代理下载失败有可读错误与自动化覆盖。
