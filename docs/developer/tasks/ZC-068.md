## 范围

- 提供 topology/角色的脱敏只读目录，显示名称、版本、角色摘要、所需能力、来源和当前不可用原因。
- `/agent` 打开 SearchPicker，参数预填搜索；确认后只对未绑定的新 thread 生效，已绑定 thread 显示当前拓扑并引导新建或 fork。
- 把选择交给现有 Runtime Profile/ModelRouter 校验；所有实际工具、审批、取消和恢复继续由既有策略层处理。

## 非范围

- 不实现任意用户 DAG、Agent Team、自由 Worker、运行中 topology 变更或自定义 Prompt 编辑。
- 不复制或显示 Architect/Executor 的内部交接正文。

## 验收清单

- Single Agent 兼容路径、合法/非法 topology、角色模型能力不足、已绑定 thread 与活动 run 均有端到端回归。
- 选择某 topology 后 Profile Key 与实际角色路由一致，旧 Runtime 按既有 DRAINING 语义处理。
- 详情与诊断不含模型凭据、完整系统 Prompt 或内部 handoff payload。
