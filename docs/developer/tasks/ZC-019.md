## 范围

扩展 ZC-035 定义的 TOML v1 Profile 配置和 run.start 参数以选择企业模型 Profile；API Key 优先通过环境变量引用，用户级 `~/.harness/config.toml` 可使用明文降级值，加载时自动收紧为 `0600`，继续只支持 OpenAI 兼容接口。

将选中的模型 Profile 的稳定、脱敏配置纳入 RuntimeProfileKey；不得因 thread 选择创建或永久缓存一张完整 Agent 图。

## 非范围

不接入 Anthropic、Gemini、Ollama 等额外协议，不实现账号登录或运行中热切换模型。

## 验收清单

- 配置优先级、默认 Profile、未知 Profile 和缺失凭据有单元测试。
- 选中的 Profile 传给模型工厂，并在初始化/线程摘要中仅以脱敏信息展示。
- 已绑定线程恢复时使用其已记录的 Profile；不同 Profile 覆盖必须以不可变绑定错误拒绝。
