## 范围

扩展 TOML 配置和 run.start 参数以选择企业模型 Profile；API Key 仍只能通过环境变量引用，继续只支持 OpenAI 兼容接口。

## 非范围

不接入 Anthropic、Gemini、Ollama 等额外协议，不实现账号登录或运行中热切换模型。

## 验收清单

- 配置优先级、默认 Profile、未知 Profile 和缺失凭据有单元测试。
- 选中的 Profile 传给模型工厂，并在初始化/线程摘要中仅以脱敏信息展示。
- 恢复线程时默认使用其已记录的 Profile，除非用户显式覆盖。
