## 范围

- 实现 Builtin、Skill、可信 MCP Prompt、已启用 Plugin Command 与受控 Prompt Template Loader，并将 descriptor 归一化进入 Registry。
- 定义来源标识、完整限定名、Builtin 保留名和冲突拒绝规则；短名称只在全局唯一时可调用，继续支持 `/skill:<canonical-id>`。
- 实现 `/reload` 范围选择与确认，只允许空闲状态构建新 snapshot；失败时回退旧 snapshot，成功后仅供后续新 thread 使用。
- 将 Plugin/MCP/Skill 信任、capability、企业策略和禁用状态映射到动态命令的隐藏/禁用结果及 TUI 来源标签。

## 非范围

- 不启动目录监听、热替换既有 PromptEpoch、自动下载插件/Skill、执行未信任 Prompt 或开放模型可调用的管理命令。
- 不放宽 MCP、Plugin、Skill 的既有完整性、认证、沙箱或项目可信边界。

## 验收清单

- 同名 Builtin/动态项、多个动态来源、禁用项、恶意 descriptor、Plugin/MCP 失效和 reload 失败均有确定解析与 UI 回归。
- reload 期间的 run、待处理审批、已有 thread 和恢复 thread 保留旧视图/PromptEpoch；新 thread 仅在成功后看到新 catalog。
- 所有动态命令来源、帮助、菜单和审计摘要一致，且不包含正文、凭据、完整 URL 或路径穿越信息。
