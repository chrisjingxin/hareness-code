## 范围

参考 dcode 的 `fetch_url` 安全策略提供单一网页读取工具，并纳入现有审批和审计机制。

## 非范围

不实现通用 Web 搜索、浏览器自动化、下载文件、Cookie、代理继承或内网访问。

## 验收清单

- URL scheme、IP 范围、DNS 解析、每跳重定向和响应大小都有测试。
- HTML 输出经过确定性 Markdown 转换和字符数截断。
- 工具调用受审批模式和 ZC-021 审计接口约束。
