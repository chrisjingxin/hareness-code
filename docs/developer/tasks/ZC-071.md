## 范围

- 定义只读配置详情、允许修改的用户级字段、不可修改原因、重启/新 thread 生效范围和预览 diff 的脱敏 DTO。
- 为白名单字段实现 compare-and-swap、完整解析/校验、managed policy lock、可信项目边界、原子写入与失败恢复；所有操作记录安全审计摘要。
- 明确拒绝 API Key、秘密 Header、任意环境变量、远端 Provider 工厂、受管锁定字段和未经信任项目配置的交互式写入。
- 为后续 TUI 与正式 CLI 复用同一 Domain Service，不以任意 `/config set <key> <value>` 作为公开接口。

## 非范围

- 不实现 Settings UI、模型 Picker、企业登录、配置同步或项目配置自动信任。
- 不改变现有 TOML 优先级、API Key 降级规则或审批模式的真实执行语义。

## 验收清单

- 合法修改、并发版本冲突、语法错误、磁盘写入失败、权限错误、managed lock 和未信任项目均有失败闭合回归。
- 预览、审计和诊断不显示秘密；损坏/失败后原配置仍可被加载。
- TypeScript/Python RPC 契约、配置迁移与用户级文件权限测试通过。
