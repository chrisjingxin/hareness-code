## 范围

限制 `ShellAllowListMiddleware` 的输入为可安全判断的单命令形式，并对绝对命令路径、参数和 shell 特殊语法建立明确策略。

## 非范围

不实现完整 shell 解释器、命令沙箱、网络策略或工作区文件路径边界。

## 验收清单

- `allowed && forbidden`、管道、重定向、`$()`、反引号和换行拼接被拒绝。
- 白名单内的单一编译、测试和 Git 只读命令仍可执行。
- 畸形引号或无法解析的输入 fail closed，且测试不调用真实危险命令。
