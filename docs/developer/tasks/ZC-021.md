## 范围

在 Python Agent 内核定义与具体日志平台无关的审计 sink 协议，并在工具和交互生命周期发送结构化事件。

## 非范围

不连接企业日志服务、不实现任意 hook 子进程、不保存完整对话内容或扩大 JSON-RPC 协议。

## 验收清单

- 写入、shell、审批、拒绝、取消和异常分别产生可关联事件。
- 脱敏和大输出截断在 sink 前完成。
- fake sink 测试验证事件顺序、关联字段和 sink 失败不影响 Agent 终态。
