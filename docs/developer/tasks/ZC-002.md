## 范围

默认将 `--cwd` 解析出的目录作为本机文件工具可访问边界。为后续显式附加目录能力预留接口，但本任务不开放绕过方式。

## 非范围

不修改 shell 命令策略、企业远端 sandbox 或 TUI 审批文案。

## 验收清单

- 读写、编辑、删除、glob、grep 的越界路径均被拒绝。
- 路径经过 canonicalize 后仍须位于工作区；symlink 不得逃逸。
- Python 测试证明所有拒绝路径均不会访问工作区外文件。
