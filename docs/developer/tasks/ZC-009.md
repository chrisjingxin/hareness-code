## 范围

交付 Harness Code 的第一个 za38 垂域能力：生成后端 Agent 骨架。模板由仓库内版本化资产提供，生成器输出必须可预测。

## 非范围

不生成前端 UI、部署配置、MCP Server、企业鉴权实现或业务领域逻辑。

## 验收清单

- 生成前预览文件清单；冲突文件默认拒绝并提示用户。
- 生成目录位于当前工作区，路径穿越和 symlink 逃逸被拒绝。
- 生成产物的最小测试、模板版本和用户文档有回归覆盖。
