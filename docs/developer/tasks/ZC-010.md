## 范围

把已解析的 `approval_mode` 从配置层传入 Agent 工厂，定义并实现 `plan`、`ask`、`auto-edit` 的稳定语义。

## 非范围

不新增永久批准、每工具白名单、企业权限中心或远端 sandbox Provider。

## 验收清单

- `ask` 对受控写入和 shell 操作发起 HITL 请求。
- `auto-edit` 只自动放行文件编辑，shell 和高风险操作仍需审批。
- `plan` 禁止实际变更工具执行，并向模型返回可操作的计划模式结果。
- Python 与 Bun 测试同时断言后端行为和 TUI 展示一致。
