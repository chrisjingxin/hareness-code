## 范围

- 扩展 `context.compact` 的版本化契约，使其可接收长度受限、脱敏处理的可选 `focus`；Python 手动压缩器仅将其作为摘要保留提示。
- 实现 `/context` Viewer，展示已知上下文窗口、估算分类、自动阈值、归档/摘要状态与最近压缩结果；不支持或不能可靠估算的项明确显示未知。
- `/compact` 无参数打开确认 Dialog，有参数预填 focus；完成后显示实际前后估算、节省和保留的最近轮次，拒绝/失败不改写当前历史。
- 将查看归档历史的入口限制为当前 project/thread 的只读虚拟路径，维持既有 Workspace Boundary。

## 非范围

- 不改变自动压缩阈值、最近两轮保留规则、熔断器或 Provider 缓存策略。
- 不向客户端发送完整 Prompt、原始大工具结果或跨 thread 归档。

## 验收清单

- 协议生成、TypeScript/Python 参数校验、空闲 thread 限制、focus 长度/恶意输入和压缩回归均通过。
- Inspector 在无 thread、无估算、压缩失败和已归档历史时给出准确的只读状态。
- focus 只影响目标 thread 的手动压缩，服务重启、并发 thread 和取消不会串线或改写错误历史。
