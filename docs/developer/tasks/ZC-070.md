## 范围

- `/status` 显示当前 project 摘要、thread、Runtime Profile 摘要、模型/拓扑、上下文、审批模式、执行后端、RuntimePool、可用 Skill/MCP 摘要及协议/sidecar 状态；字段受 capability 限制。
- `/doctor` 聚合配置来源与校验、SQLite/权限、协议协商、Provider 可用性、OpenTUI 资源、Skill catalog 与已接入 MCP 的结构化诊断，分为 OK、Warning、Error、Manual action。
- 为客户端增加只读、脱敏的诊断 DTO/RPC 或复用既有读取路径；没有对应后端能力时显示 not supported。
- 将 `version` 作为低频兼容信息收敛进 status/doctor，保留 Slash 别名但不占默认推荐位置。

## 非范围

- 不在 `/doctor` 中安装依赖、修改配置、重连 MCP、发送网络探测或自动修复。
- 不实现 Usage 统计、Tool Explorer 或 Settings 编辑。

## 验收清单

- status/doctor 均不创建 run，脱敏测试覆盖 API Key、Header、原始路径、Prompt 与完整工具输出。
- 数据库损坏、未配置模型、协议不兼容、catalog/MCP 不可用和不支持字段均有稳定视图与错误码映射。
- OpenTUI、TypeScript/Python 契约和文档回归通过。
