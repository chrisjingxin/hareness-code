## 范围

把已验证的 stdio MCP 配置接入 `langchain-mcp-adapters`，建立会话所有权、工具过滤和 server 状态 DTO。

## 非范围

不实现 HTTP/SSE transport、OAuth、项目未信任配置的临时批准 UI 或 MCP 热重载。

## 验收清单

- mock stdio server 的工具能注册、调用和在退出时清理。
- 不可执行命令、启动超时、工具 schema 错误和 Agent 创建失败都有诊断测试。
- 仅已信任且通过过滤的工具进入模型可见工具集。
