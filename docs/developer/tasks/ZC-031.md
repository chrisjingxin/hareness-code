## 范围

复用 `unicode_security.py` 的检测结果，为审批请求补充结构化风险信息和安全显示文本。

## 非范围

不改写用户输入、不阻止纯文本回复、不实现通用内容审查或新增协议 major 版本。

## 验收清单

- shell 命令、文件路径和 MCP 来源中的危险 Unicode 能被检测和标记。
- TUI 在批准前展示风险类型及标记后的预览，不因未知字符崩溃。
- 正常 Unicode 文件名和中文内容不会产生误报或被破坏。
