## 范围

只完成配置发现和校验，为后续信任与 transport 加载提供已验证的配置输入。

## 非范围

不创建 MCP 连接、不注册工具、不写入项目级信任记录、不实现 OAuth 或 TUI 状态页。

## 验收清单

- 用户级、项目级、显式配置的优先级和格式错误有测试。
- 项目配置仅作为未执行的数据返回给后续信任阶段。
- 环境变量引用只验证名称与声明，不读取、插值或回显秘密。
