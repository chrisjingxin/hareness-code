## 范围

支持当前协议中的多题 `question` 请求。保留单题行为，不改变审批交互、Slash 菜单或协议 Schema。

## 非范围

不实现问卷条件逻辑、多选题、文件选择器或新的 Agent 工具。

## 验收清单

- 两题及以上的请求可依次作答、返回、提交和取消。
- 返回值使用问题 ID，而非问题文本或显示顺序作为协议键。
- Bun TUI 测试覆盖选择、自由输入、取消和运行恢复。
