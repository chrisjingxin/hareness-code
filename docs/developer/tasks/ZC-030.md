## 范围

消费 ZC-005 的已校验配置，维护用户级 trust store，并为未来 TUI 审批和 CLI 管理命令提供稳定服务接口。

## 非范围

不加载 MCP transport、不执行项目配置中的命令、不插值环境变量、不实现 OAuth。

## 验收清单

- 首次项目配置默认不可信，批准后在内容不变时可信。
- 文件内容、项目根目录或存储版本变化会使信任失效或要求重新确认。
- trust store 使用原子写入、受限文件权限，并且损坏时 fail closed。
