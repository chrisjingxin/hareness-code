## 范围

参考 dcode 的 LocalContext 思路，生成只读、可缓存、可测试的工作区摘要，并把它加入 Agent 的模型上下文。

## 非范围

不执行项目安装、构建或测试命令；不上传 Git 信息；不加入 MCP、模型凭据或完整目录树。

## 验收清单

- Git 分支、脏状态、识别出的语言/包管理器和受限目录摘要可预测。
- 每项读取都受命令超时、输出长度和路径边界限制。
- 无 Git、无 Node/Python 或命令失败时仍可正常启动 Agent。
