## 范围

- 定义脱敏 turn 时间线与快照选择 DTO，提供 `/fork`、`/rewind` 的 TimelinePicker、影响摘要与确认 Dialog。
- fork 复制所需的可恢复历史/检查点引用并建立父子关系；rewind 将目标 thread 受控回退到选择点，记录可审计终态。
- 与 Thread Manager 共用当前 project 可见性和稳定摘要，禁止用户输入或渲染内部 `thread_id`。
- 在 replay、取消、待处理审批/提问与 Runtime 重建的情况下定义确定的拒绝或恢复语义。

## 非范围

- 不将 rewind 伪装为文件 undo，不执行 Git reset，不支持跨 project 合并、云同步或任意历史编辑。
- 不实现 Agent Team 内部 mailbox 的时间线展示。

## 验收清单

- 多分支、损坏快照、并发读取、恢复、取消和重启场景不丢失源 thread 或让子分支串线。
- rewind 前显示丢弃范围，确认/取消行为稳定；未完成交互和活动 run 无法执行。
- Timeline 不包含完整 Prompt、工具大输出、秘密或内部身份字段。
