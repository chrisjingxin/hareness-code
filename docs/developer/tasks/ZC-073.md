## 范围

- 定义 MCP/Tool 只读状态 DTO，展示连接状态、来源、信任、支持的 transport、工具/Prompt 数量、最近脱敏错误、可用性与审批风险。
- `/mcp` 在 capability 允许时提供 inspect、启停、重连和刷新等已实现动作；动作复用可信配置与生命周期服务，不直接执行任意命令或 URL。
- `/tools` 汇总 Builtin、可信 MCP 和后续 Plugin 工具的名称、输入 schema 摘要、来源、审批要求与可用性，并禁止在该页面直接调用 Tool。
- 将未实现 transport 或未协商 capability 显示为 not supported，保留后续 HTTP/SSE 扩展位。

## 非范围

- 不实现 OAuth、任意 URL 添加、项目 MCP 自动信任、MCP Prompt 命令加载或 Tool 的手工执行控制台。
- 不显示 Header、完整 URL 查询、凭据、原始工具结果或跨 project 配置。

## 验收清单

- stdio、HTTP/SSE（若已启用）、断连、配置无效、信任撤销、重连失败和 capability 缺失均有 TUI/协议回归。
- Tool Explorer 与真实 ToolScheduler/审批策略的风险和可用性一致，未注册来源不能伪造显示项。
- 诊断输出的敏感字段 redaction 有单元测试，窄终端与错误状态可操作。
