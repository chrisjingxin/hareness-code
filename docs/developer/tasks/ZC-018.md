## 范围

实现 MCP 授权状态机和安全 token store，并通过协议的能力协商暴露最小交互请求。

## 非范围

不为模型网关、企业 SSO 或远端 sandbox 实现认证；不支持未可信项目 MCP 的 OAuth。

## 验收清单

- OAuth 启动、成功、用户拒绝、回调失败和 refresh 失败均有 mock 测试。
- token 文件权限、原子写入和脱敏日志有测试。
- 未协商 OAuth capability 的客户端不会收到交互请求。
