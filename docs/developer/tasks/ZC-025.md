## 范围

实现 POSIX 安装脚本和版本 manifest 消费逻辑，安装到用户级目录并保留可回滚的旧版本。

## 非范围

不实现 Windows 安装器、后台自动更新、企业软件分发平台或安装后遥测。

## 验收清单

- macOS arm64 与 Linux x64 的正确 artifact 被选择和 SHA-256 校验。
- 网络、解压或校验失败时旧版本保持可用且临时文件被清理。
- shell 冒烟测试验证新终端可找到并启动 `za38`。
