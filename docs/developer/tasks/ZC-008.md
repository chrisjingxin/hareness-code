## 范围

通用本机/远端选择器、factory 契约和 fake-provider 测试已经交付。本任务仅承接企业 Provider 的具体实现。

## 非范围

不修改默认本机行为，不实现 Docker、Podman 或新的通用 sandbox 抽象。
不修改用户侧模型网关协议。

## 验收清单

- 需要企业提供 Provider API、认证方式、工作区同步/回写语义和审计字段后才能认领。
- 真实 Provider 不可用、认证失败、同步失败、取消和回写失败均不得退回本机执行。
- 集成测试在企业隔离测试环境验证文件、shell、取消和审计事件。
