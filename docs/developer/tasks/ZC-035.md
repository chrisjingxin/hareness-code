## 范围

把当前 `[model]`、`[tools]` 和 `[sandbox]` 迁移为 v1 的 `[models]`、`[approval]`、`[execution]` 结构。环境变量和 CLI `--sandbox` 继续覆盖已解析的用户配置。

## 非范围

不实现模型 Profile 运行时选择、项目配置可信、managed 配置、配置热重载、通用环境变量插值或新的 JSON-RPC 方法。

## 验收清单

- `[models.profiles.<name>]` 只允许一个默认 Profile；多个 Profile 的选择留给 ZC-019。
- 工作区 `.harness/config.toml` 或 `.harness/config.local.toml` 被拒绝，并提示移至用户配置或使用 `--config`。
- `config show` 和初始化摘要只显示脱敏信息，错误不包含环境变量值。
- Python/Bun 回归、项目检查和 mock SSE E2E 通过。
