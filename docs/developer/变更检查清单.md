# 变更检查清单

在将任务标记为完成前，逐项确认：

- [ ] 代码仅影响已认领范围，中文注释符合 [AGENTS.md](../../AGENTS.md)。
- [ ] 已新增或更新相应自动化测试，并记录可复现的测试证据。
- [ ] 若用户能感知行为、命令、配置或限制，已更新 `docs/user/`。
- [ ] 若涉及架构、协议、配置、安全边界或开发方式，已更新 `docs/developer/`。
- [ ] 任务文件记录了文档影响、关联提交或 PR；已运行 `bun run tasks:sync`。
- [ ] 已运行 `bun run project:check`、`bun run typecheck`、`bun run test`；构建变更已运行 `bun run build`。
- [ ] 若需要发版，版本只通过 `bun run version:set <SemVer>` 修改，且 `bun run release:check` 通过。

不可通过直接修改 [任务看板](任务看板.md) 或分散版本字段来绕过检查；前者由脚本生成，后者以根目录 `VERSION` 为唯一来源。
